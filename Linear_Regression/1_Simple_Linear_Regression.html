Simple Linear Regression

Simple Linear Regression is a method to model the relationship between a dependent variable (Y) and one independent variable (X) using a straight line.

Equation

Y = β₀ + β₁X + ε

  • Y – Dependent variable
  • X – Independent variable
  • β₀ – Intercept
  • β₁ – Slope
  • ε – Error term

Assumptions

  • Linearity: Relationship between X and Y is linear.
  • Independence: Observations are independent of each other.
  • Homoscedasticity: Constant variance of residuals.
  • Normality: Residuals are normally distributed.

Objective

To estimate β₀ and β₁ such that the sum of squared residuals is minimized.

Example

Suppose we want to predict a student’s score (Y) based on hours studied (X).

Given data points:

X (Hours): 1, 2, 3, 4, 5  
Y (Scores): 2, 4, 5, 4, 5

Fitted line: Y = 2.2 + 0.6X

Interpretation: For each additional hour studied, the score increases by 0.6 on average.
